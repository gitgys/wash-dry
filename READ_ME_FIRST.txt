Here's what to do:

STEP 1: Install Node.js
   → Go to nodejs.org
   → Download & install (5 min)

STEP 2: Get Website Running
   In terminal:
   $ cd [project folder]
   $ npm install -g pnpm
   $ pnpm install
   $ pnpm dev
   
   Then open: http://localhost:3000

STEP 3: Update Your Info

STEP 4: Deploy to Vercel

   $ vercel login
